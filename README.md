# JSHunt API com NodeJS, MongoDB e Express

Uma simples API NodeJS com CRUD e Paginação

## Após baixar o Projeto

Lembre-se de rodar o comando:

```
npm install
```

ou somente:

```
yarn
```

### Prerequisitos

NodeJS versão > 10 e Yarn instalados globalmente
